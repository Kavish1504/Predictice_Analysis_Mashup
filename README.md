# Title: Mashup

# 1. Methodology

Given videos Collection based on singer name ----> Converting videos to audios ---> Trimming the audio to given length ----> Merging all the trimmed audios to one file

# 2. Description

- Data: Youtube Videos
- N: No. of videos
- Y: Length of each video
- Output: mashup.mp3

# 3. Objective

- To automatically download songs of a selected singer from online sources.
- To extract, trim, and merge audio clips into a single mashup file.
- To compress the final mashup into a ZIP file.
- To deliver the mashup to the user via email.
